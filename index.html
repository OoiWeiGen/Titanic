4. Data Cleaning

Identify and handle missing values in the dataset (e.g., fill missing age values with median).

 

Perform any necessary data transformations or cleaning steps.

 

Python command :

# Identify the missing values

print('Number of missing values per column:')

print(df.isnull().sum())

 

# Calcualate the median age

median_age = df['Age'].median()

print('Median age before missing values:', median_age)

 

# Fill missing age values with the median age

df['Age'] = df['Age'].fillna(median_age)

Output of the command :

 

Explain what the command does :

 

1. print(df.isnull().sum()): This command calculates the total number of missing values in each column of the DataFrame `df` and prints the result.

     - `df.isnull()`: This part generates a DataFrame of the same shape as `df` where each cell contains a boolean value indicating whether the corresponding cell in `df` is null or not.

     - `.sum()`: This calculates the sum of the boolean values along each column, resulting in a Series that displays the total count of missing values in each column.

 

2. median_age = df['Age'].median()`: This line calculates the median age from the 'Age' column of the DataFrame `df`

 

3. df['Age'] = df['Age'].fillna(median_age)`: This command fills the missing values in the 'Age' column of the DataFrame `df` with the calculated median age.

     - `df['Age'].fillna(median_age)`: This fills the missing values in the 'Age' column with the calculated median age, generating a new Series.

     - `df['Age'] = ...`: This assigns the new Series back to the 'Age' column of the DataFrame `df`, replacing the original 'Age' column with the one where missing values are filled.

 

Overall, the code first identifies and reports the number of missing values in each column, then calculates the median age from the 'Age' column, and finally replaces missing values in the 'Age' column with the calculated median age.

 

Explain what the command mean :

 

1.print(df.isnull().sum()): This calculates and prints the number of missing values in each column of the DataFrame `df`. The result shows how many missing values are there in each column.

 

2. median_age = df['Age'].median. It selects the 'Age' column from the DataFrame `df` and computes the median value of that column.

 

3. df['Age'] = df['Age'].fillna(median_age): This command fills in missing values in the 'Age' column of the DataFrame `df` with the calculated median age. It selects the 'Age' column, fills the missing values with the median age (`median_age`), and updates the 'Age' column in the DataFrame with these filled values.

6. Data Visualization

Create a bar chart to visualize the survival rate by passenger class.

 

Add appropriate labels and a title to the chart.

Python command :

 

 

 import matplotlib.pyplot as plt

import pandas as pd

 

# Calculate the mean survival rate by passenger class

survival_rates = df.groupby('Pclass')['Survived'].mean()

 

# Create a bar chart

plt.figure(figsize=(8, 6))

bar = plt.bar(survival_rates.index, survival_rates.values)

 

# Change the color of each bar

bar[0].set_color('red')    # First class

bar[1].set_color('green')  # Second class

bar[2].set_color('blue')   # Third class

 

# Add title and labels

plt.title('Survival Rate by Passenger Class')

plt.xlabel('Passenger Class')

plt.ylabel('Survival Rate')

plt.xticks(survival_rate.index,['First class','Second class','Third class'])

 

# Display the plot

plt.show()

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Output of the command:

Explain what the command does : Let's break down the commands:

 

1. Calculate the mean survival rate by passenger class:

   - `survival_rates = df.groupby('Pclass')['Survived'].mean()`: This line calculates the mean survival rate for each passenger class by grouping the DataFrame `df` by the 'Pclass' column and then calculating the mean of the 'Survived' column within each group. The result is stored in the `survival_rates` Series.

 

2.*Create a bar chart:

   - `plt.figure(figsize=(8, 6))`: This command initializes a new figure for the plot with a specific size of 8x6 inches.

The x-values are taken from the index of the `survival_rates` Series (which corresponds to the passenger class), and the y-values are taken from the values of the `survival_rates` Series.

   - `bar[0].set_color('red')`: This sets the color of the first bar (representing the first class) to red.

   - `bar[1].set_color('green')`: This sets the color of the second bar (representing the second class) to green.

   - `plt.title('Survival Rate by Passenger Class')`: This adds a title to the plot, specifying that it shows the survival rate by passenger class.

   - `plt.xlabel('Passenger Class')`: This adds a label to the x-axis, indicating the passenger class.

   - `plt.ylabel('Survival Rate')`: This adds a label to the y-axis, indicating the survival rate.
